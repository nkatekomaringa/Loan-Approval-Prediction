# Loan-Approval-Prediction
A beginner level binary classification projecct that predicts whether a home-loan application gets approved based on demographic and financial featurs.

#Problem Statement
Given an applicants gender, marital status, dependents, educations, employments, income, request loan amount, term, credit history, and property are, predict whether the following loans will be approved(Loan_Status = Y or N).

#Dataset
-Source: https://www.kaggle.com/datasets/altruistdelhite04/loan-prediction-problem-dataset?resource=download
-Samples: 614 applications
-Features: 11 inputs + 1 binary target
